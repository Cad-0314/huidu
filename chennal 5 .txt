Generate signature
I. Combined signature data format, type FORM-DATA (
Content-Type:multipart/form-data;
)


{
    "app_key":"到后台应用管理查看app key",
    "sign_ver":"1.0", // 固定值
    "timestamp":"当前系统时间戳秒级（10位）",
    //其他参数...
    "sign":"拼接参数字符串后md5加密值，需要注意小写",
}
2. All parameters are sorted in ascending order by field name, combined into the raw HTTP string, and finally appended with app_secret. The combined string is then encrypted using MD5.
Empty strings are not included in the signature; the value 0 is required to be included in the signature.


app_key=xxxx&...&sign_ver=1.0&...&timestamp=128097733&...&app_secret=xxxxxxxxxx


III. Generate MD5 HTTP raw string.




http原始字符串: app_key=e9c5c0a04c&sign_ver=1.0&timestamp=1731984499&app_secret=YzM4NzIxNjc3NjhiOGQ4MjFjNjk5ODBkYmE5NWQ3ZDA0MzUxNGJlMWEyZDA0MDkyZDNjNTgxNzIzODc1OTlhZA==


md5加密字符串: 41024c68460fab5f237930ec8cf38416
